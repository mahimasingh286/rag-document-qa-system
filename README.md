# RAG Document Q&A System

A Retrieval-Augmented Generation (RAG) system that allows users to 
upload PDF documents and ask questions, getting accurate answers 
using open-source LLMs.

## 🛠️ Tech Stack
- Python, Flask
- Hugging Face Transformers
- ChromaDB (Vector Database)
- Sentence Transformers

## 🚀 Features
- Upload any PDF and ask questions in natural language
- Semantic search using vector embeddings
- Sub-2 second query response time on 50+ page documents
- REST API built with Flask

## ⚙️ How to Run

1. Clone the repository
   git clone https://github.com/mahimasingh286/rag-document-qa-system.git

2. Install dependencies
   pip install -r requirements.txt

3. Run the app
   python app.py
